To switch the API, update the app_settings table directly:

key	value options
airtime_api	vtpass · clubkonnect · autopilot · 

To activate a provider for a network, set data_api_{network_key} in app_settings to easyaccess, aabaxztech, globacom (for glo alone) or legitdataway, then fill in the API tokens and the plan *_id columns.
